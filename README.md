# worktree-manager

> Manage git worktrees for multiple repositories easily.

## Install

```
$ npm install worktree-manager
```

## Usage

```js
import worktreeManager from 'worktree-manager';

worktreeManager('unicorns');
//=> 'unicorns & rainbows'
```
